# Contributing

Thanks for your interest in contributing!

## Getting Started
- Fork the repository and create a new branch.
- Make focused, minimal changes.
- Ensure your changes build and run locally.

## Development
- Install dependencies: `npm install`
- Start dev server: `npm run tauri dev`

## Pull Requests
- Describe the problem and the solution clearly.
- Include screenshots or logs when applicable.
- Keep PRs small and focused.

## Code Style
- Follow existing formatting and structure.
- Avoid unnecessary dependencies.

## Reporting Issues
- Use the issue templates when available.
- Include steps to reproduce, expected/actual behavior, and environment details.
